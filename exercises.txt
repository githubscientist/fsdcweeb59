FUNCTIONS

1. Greeting: Write a function greet(name) that returns "Hello, <name>!".

2. Calculator: Write four functions — add(a, b), subtract(a, b), multiply(a, b), divide(a, b) — that return the result of the operation.

3. Even or Odd: Write a function isEven(n) that returns true if the number is even, false otherwise.

4. Max of Two: Write a function maxOfTwo(a, b) that returns the larger of the two numbers.

5. Area of a Rectangle: Write a function rectArea(length, width) that returns the area.

6. Temperature Converter: Write a function celsiusToFahrenheit(c) that converts Celsius to Fahrenheit using the formula (c * 9/5) + 32.

---

ARRAYS

1. First and Last: Given an array, access and print the first and last element.

2. Add Items: Create an empty array and use push() to add 3 fruits to it. Print the result.

3. Remove Last: Given an array of 5 numbers, use pop() to remove the last element and print the updated array.

4. Array Length: Write a function getLength(arr) that returns the number of elements in an array.

5. Combine Arrays: Given two arrays [1, 2, 3] and [4, 5, 6], use concat() to merge them into one array.

6. Check Membership: Write a function contains(arr, item) that returns true if the item exists in the array using indexOf() or includes().
